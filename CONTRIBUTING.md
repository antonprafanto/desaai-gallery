# Contributing to DesaAI

DesaAI was created for the [Gemma 4 Impact Challenge](https://www.kaggle.com/competitions/gemma-4-good-hackathon). After the competition, we welcome community contributions to expand DesaAI's reach to more rural communities in Indonesia and beyond.

## Ways to Contribute

- **New Skills** — Add new Agent Skills for other rural use cases (healthcare, weather, livestock, etc.)
- **Language Support** — Add more regional language dictionaries (Sundanese, Batak, Bugis, etc.)
- **Bug Reports** — Open a [GitHub Issue](https://github.com/antonprafanto/desaai-gallery/issues)
- **Translations** — Help translate the app UI to Bahasa Indonesia or regional languages

## Skill Development

See [skills/README.md](skills/README.md) for the full guide on creating custom skills.

## Code of Conduct

Be respectful and constructive. This project serves rural Indonesian communities — keep that mission in mind.
